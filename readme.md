# Projekt

## Introduction
Dashboard that checks the weather and compares the weather in different places.

## Installation
1. install npm:
``` 
sudo yum install npm 
```

2. Clone project
``` 
git clone url
```

3. Install dependiences
``` 
cd project
npm i
```
4. start application
```
npm start
```
